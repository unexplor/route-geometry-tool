# 测试数据集说明

本目录提供 7 组 CSV 测试数据，覆盖线路平面几何的各种场景。所有数据均经过几何可行性验证（切线长 < 交点间距、偏角 > 缓和曲线角之和），可直接导入工具构建线路。

CSV 列顺序：`里程, X坐标, Y坐标, 前缓长, 半径, 后缓长`（首行为表头，导入时自动识别）。

## 数据集一览

| 文件 | 场景 | 交点数 | 段数 | 说明 |
|------|------|--------|------|------|
| `01_straight_line.csv` | 纯直线 | 3 | 2 | 半径全为0，验证直线段与里程插值 |
| `02_single_left_curve.csv` | 单条左转曲线 | 3 | 5 | R=300，标准对称缓和曲线，左转90° |
| `03_single_right_curve.csv` | 单条右转曲线 | 3 | 5 | R=400，右转，验证Y负方向偏转 |
| `04_asymmetric.csv` | 不对称缓和曲线 | 3 | 5 | 前缓30/后缓80，R=500，l1≠l2 |
| `05_no_transition.csv` | 无缓和曲线 | 3 | 3 | l1=l2=0，只有圆曲线 |
| `06_s_curve.csv` | S形双曲线 | 4 | 9 | 左转+右转，验证反向偏转衔接 |
| `07_complex.csv` | 复杂线路 | 5 | 13 | 3条曲线（左-右-左），综合场景 |

## 场景详细说明

### 01 纯直线
最简单场景，全部半径为0。用于验证直线段方位角计算和里程-坐标线性关系。预期：3个交点生成2条直线段，里程范围 [0, 2500]。

### 02 单条左转曲线（对称）
标准场景：起点 → 左转90°曲线 → 终点。前缓=后缓=50m，R=300。验证缓和曲线（回旋线）级数展开和坐标变换的正确性。预期：5段（直线+前缓+圆曲+后缓+直线），曲线向 Y 正方向（左侧）偏。

### 03 单条右转曲线
与02对称，但向右偏转。验证 `turn_sign=-1` 的坐标变换（Y 取反）。预期：中点坐标 Y 为负值（曲线向 Y 负方向偏）。

### 04 不对称缓和曲线
前缓和后缓长度不同（30 vs 80），R=500。验证不对称曲线的 T1≠T2 计算、ZH/HY/YH/HZ 四个关键点的正确推导。

### 05 无缓和曲线
l1=l2=0，只有直线和圆曲线直接相切。验证边界情况：无缓和曲线段生成，圆曲线起点=ZH=HY。预期：3段（直线+圆曲+直线）。

### 06 S形曲线
左转曲线接右转曲线，中间有直线段。验证两条反向曲线的衔接、里程连续性、坐标连续性。预期：9段，包含2条圆曲线。

### 07 复杂线路
3条曲线（左-右-左）+ 4条直线段，5个交点。综合验证多曲线场景的段落生成、里程单调性、段间连续性。预期：13段。

## 几何约束说明

设计测试数据时必须满足以下约束，否则工具构建时会报错：

1. **切线长 < 交点间距**：`T = (R+p)·tan(α/2) + m` 必须小于相邻交点距离，否则 ZH/HZ 点会越过相邻交点。
2. **偏角 > 缓和曲线角之和**：`α > β1 + β2`（其中 `β = Ls/(2R)`），否则圆曲线长 Ly < 0。
3. **缓长 ≤ π·R**：缓和曲线长不能超过半圆周长。
4. **半径 ≥ 0，缓长 ≥ 0**：输入合法性。

> ⚠️ **常见陷阱**：半径 R=800 配 500m 交点间距在 90° 偏角下不可行（切线长 ≈ 825 > 500）。设计大偏角曲线时需相应增大交点间距或减小半径。

## 使用方法

### 方式一：GUI导入
运行 `python route_geometry_tool/main.py`，点击「导入CSV」选择文件，或点击「加载示例」加载内置数据。

### 方式二：命令行验证
```bash
python -c "
import sys; sys.path.insert(0,'.')
from route_geometry_tool.utils.csv_handler import load_csv_file
from route_geometry_tool.core.route_builder import RouteBuilder
from route_geometry_tool.core.query import RouteQuery

jds = load_csv_file('route_geometry_tool/examples/06_s_curve.csv')
segs = RouteBuilder(jds).build()
q = RouteQuery(segs)
print('段数:', len(segs))
print('里程范围:', q.min_mileage, '~', q.max_mileage)
print('中点查询:', q.query((q.min_mileage + q.max_mileage)/2))
"
```
