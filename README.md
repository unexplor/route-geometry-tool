# 铁路线路平面几何计算工具

成都云幽科技有限公司 编程作业

## 功能

1. **交点输入**: 表格录入（里程、X/Y坐标、前缓长、半径、后缓长），支持增删改、CSV导入导出
2. **线路构建**: 根据交点自动生成直线段、缓和曲线段（回旋线）、圆曲线段
3. **里程查询**: 输入任意里程，输出坐标、切线方向角、切线向量、法线向量
4. **批量查询**: 起止里程 + 步长
5. **图形绘制**: 线路平面示意图，不同线型不同颜色，悬停显示里程坐标

## 技术栈

- Python 3.10+（仅标准库，无第三方依赖）
- GUI: Tkinter
- 绘图: Tkinter Canvas

## 运行

```bash
python route_geometry_tool/main.py
```

## 设计说明

### 架构

模块化分层：core/（纯数学，零UI依赖）、ui/（界面）、utils/（工具）。核心计算与界面完全分离，便于单元测试。

### 缓和曲线模型（回旋线 / Clothoid）

曲率随弧长线性变化 κ(l) = l/(R·Ls)。以曲率0端为原点，级数展开取前3项：

```
x(l) = l - l⁵/(40R²Ls²) + l⁹/(3456R⁴Ls⁴)
y(l) = l³/(6RLs) - l⁷/(336R³Ls³) + l¹¹/(42240R⁵Ls⁵)
θ(l) = l²/(2RLs)
```

通过局部→全局坐标变换（含左转/右转符号）统一处理三种曲线段。

### 切线长（基本型平曲线）

内移距 p = Ls²/(24R)、切垂距 m = Ls/2 − Ls³/(240R²)（取一阶近似）。
切线长须同时满足圆心"到入射切线距离 R+p₁、到出射切线距离 R+p₂"两个约束，解得：

```
T₁ = (R+p₁)·tan(α/2) + m₁ + (p₂−p₁)/sin α
T₂ = (R+p₂)·tan(α/2) + m₂ + (p₁−p₂)/sin α
```

修正项 (p₂−p₁)/sin α 在对称曲线（p₁=p₂）时为 0，退化为对称公式
T=(R+p)·tan(α/2)+m。漏掉该项会使 l₁≠l₂ 时 YH 不落在圆曲线上
（偏差可达米级），此前即由此对称用例掩盖。

### 线路构建流程

1. 遍历中间交点，计算入射/出射方位角、偏角α
2. 计算缓和曲线参数 β、内移距 p、切垂距 m、切线长 T
3. 确定 ZH、HY、QZ、YH、HZ 五个关键点坐标和里程（QZ 为圆曲线弧长中点，里程 = HY里程 + Ly/2，坐标取圆弧上 s=Ly/2 处）
4. 按里程顺序生成段：直线 → 前缓和 → 圆曲线 → 后缓和 → 直线

### 法线方向

法线 = 切线逆时针旋转 90°：normal = (-sin α, cos α)

## 边界处理

- 半径=0 → 纯直线，不生成曲线段
- 缓长=0 → 无缓和曲线，只有圆曲线
- 里程超出范围 → 报错提示
- 输入验证：半径≥0，缓长≥0，缓长 ≤ π·R
- 几何约束：偏角必须大于缓和曲线角之和（否则报错提示增大半径）
